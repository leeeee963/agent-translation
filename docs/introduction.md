# AgentTranslation — 多语种智能翻译工具

一键启动，拖拽上传文件，选择目标语言，即可获得接近人工译员质量的翻译结果。

## 核心能力

- **15 种语言互译** — 中、英、日、韩、法、德、西、葡、俄、越、泰、印尼、蒙、哈萨克等
- **支持多种文件格式** — PPT、Word、Markdown、HTML、字幕（SRT/VTT/ASS）、本地化文件（JSON/YAML/PO/XLIFF）
- **术语管理** — AI 自动提取专业术语，人工审核后全文一致性翻译，支持术语库跨项目复用
- **自然度审校** — 翻译完成后 AI 二次润色，优化目标语言的表达习惯和流畅度
- **多语言并行** — 多个目标语言同时翻译，无需排队等待

## 使用方式

Mac 终端运行 `bash start.sh`，Windows 双击 `start.bat`，浏览器自动打开即可使用。首次启动自动安装所有依赖，无需手动配置环境。
